# 📚 推荐学习材料

## AI 基础

### LLM 核心概念

- [What is a Large Language Model?](https://www.youtube.com/watch?v=5sLYAQS9sWQ)（视频）：建立 LLM 最小必要直觉
- [Hugging Face LLM Course Chapter 1](https://huggingface.co/learn/nlp-course/chapter1/1)：系统理解 LLM 工作原理

### API 调用入门

- [Anthropic: Building with the Claude API](https://docs.anthropic.com/claude/docs/building-with-claude)：官方 API 完整入门课程
- [Z.ai API 开发者文档](https://z.ai/docs)：GLM MaaS API 入门，OpenAI 兼容接口，5 分钟跑通第一个请求
- [Z.ai Coding Plan](https://z.ai/coding-plan)：解锁 GLM 全系列模型调用

### AI Coding 工具

- [Claude Code 101](https://docs.anthropic.com/claude-code)：AI coding 工具快速上手
- [Cursor 官方文档](https://docs.cursor.com/)：AI 原生 IDE 使用指南

### Agent 基础

- [Microsoft《AI Agents for Beginners》](https://github.com/microsoft/ai-agents-for-beginners)：建立 agent 整体直觉
- [OpenAI Agents SDK Intro](https://platform.openai.com/docs/guides/agents)：理解 agent 框架
- [LangGraph Overview](https://langchain-ai.github.io/langgraph/)：理解 agent 的组织方式
- [Hermes Agent Docs](https://hermes-agent.dev/docs)：理解代理、工具调用、技能、记忆和长期执行

---

- [OpenAI Text Generation Guide](https://platform.openai.com/docs/guides/text-generation)：了解 LLM 在 API 中如何接收输入、生成文本和返回结构化内容。
- [OpenAI Tokenizer](https://platform.openai.com/tokenizer)：直观看不同文本、代码、地址和 JSON 会被切成多少 token。
- [OpenAI Embeddings Guide](https://platform.openai.com/docs/guides/embeddings)：理解 embedding 在搜索、聚类和 RAG 中的基础作用。
- [Attention Is All You Need](https://arxiv.org/abs/1706.03762)：Transformer 的原始论文，适合建立 attention 和现代大模型架构的技术背景。
- [OWASP Top 10 for LLM Applications](https://owasp.org/www-project-top-10-for-large-language-model-applications/)：从安全角度理解 Prompt Injection、敏感信息泄露、过度代理等常见风险。
